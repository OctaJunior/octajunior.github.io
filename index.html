📚 Curriculum

What Your Child Will Build & Learn

16 live sessions across 8 weekends. Each session builds on the last — from zero to a working robot!

🔌

Electronics & Circuits

Build real circuits with LEDs, buzzers, motors and sensors using BBC Micro:bit. Understand how electronics work, hands-on.

💻

Python Programming

Start from visual block coding and graduate to real Python. Write programs that make hardware respond to your commands.

🤖

Build a Robot Car

Assemble a wheeled robot car from scratch and program it to move, turn, and navigate autonomously using sensors.

🧠

Artificial Intelligence

Train a real Machine Learning model for gesture recognition. Understand how AI makes decisions — not just theory, but in practice.

🏆

Personal Innovation Project

Every child defines a problem and builds a solution. Presented on Demo Day to family, mentors and peers — real confidence builder!

📜

Verified Certificate & Portfolio

Earn a verified digital certificate upon completion. Perfect for school applications, college portfolios, and STEM competitions.
